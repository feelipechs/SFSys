# SFSys
Sistema ONG Sem Fome | Unoeste


### Como colaborar com o projeto

Primeiramente, será necessário criar um banco de dados e conectá-lo ao backend, conforme tutorial do github, pois ainda não foi feito o deploy na nuvem.
Depois de instalar as dependências de ambos os sides (back e front), rode as migrations e seeds, respectivamente
```
npm run db:migrate
npm run seed
```
Inicie as aplicações:
```
backend: npm run dev
frontend: npm run dev
```
Pronto, basta iniciar o desenvolvimento.
